This directory contains compiled circuit artifacts generated by circom.

Run the following to compile:
  circom circuits/password_policy.circom --r1cs --wasm --sym -o build/

Generated files (gitignored):
  password_policy.r1cs         — R1CS constraint system
  password_policy.sym          — Debug symbols
  password_policy_js/
    password_policy.wasm       — WebAssembly witness generator
    witness_calculator.js      — JS witness computation helper
    generate_witness.js        — Entry point for witness generation

These files are excluded from git (see .gitignore) because they are
derived artifacts. Run setup.sh to regenerate them.
